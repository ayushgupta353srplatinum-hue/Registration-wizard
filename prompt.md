#  AI Prompt Reference – Multi Step Form Project

This file contains the important prompts I used to understand and solve complex parts of the project.

## Form Validation Issue (Next Button Not Working)

Even after filling all fields, the Next button was not working.

Why is my Next button not working even after filling all input fields in a React form?  
Explain how to validate multiple fields properly before moving to the next step.

##  Show / Hide Password Toggle

How can I implement a show/hide password toggle in React using useState?  
Explain how input type changes dynamically between "password" and "text".

##  Date Input Icon White in Dark Theme

The default calendar icon was not visible in dark UI.

How to change the default date input calendar icon color in CSS for dark theme?  
Explain how filter: invert() works.

##  Yellow Blur Background Effect

How can I create a blurred yellow glowing background effect in CSS?  
I want a soft gradient blur behind the card component.

Concepts involved:
- position: absolute
- filter: blur()
- radial-gradient
- z-index layering
##  Centering Card Perfectly (Flexbox)

Explain how to perfectly center a card both vertically and horizontally using Flexbox in CSS.

Concepts learned:
- display: flex
- justify-content
- align-items
- min-height: 100vh

##  Conditional Rendering Between Steps

How does conditional rendering work in React?  
Explain how to render different form steps based on state value.
